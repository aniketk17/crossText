How to Use CrossText
Getting Started
  1. Open CrossText in your browser
  2. You'll be assigned a random username automatically
  3. Check that your status shows "Connected" at the top of the page
Sharing Text
  1. Type or paste text in the Clipboard section
  2. To send to one person:
    • Hover over a user bubble in the Nearby Users area
    • Click the Paste button that appears
  3. To send to everyone:
    • Click the Broadcast to All button
Receiving Text
  • When someone sends you text, it appears in the Received Text box
  • The sender's name appears above the message
  • You can select and copy text from this box to use elsewhere
Troubleshooting
  • If disconnected, refresh the page
  • Allow location access if prompted by your browser
  • Make sure your internet connection is working
